# Phase 2：语音、图片与后台起步规格

## 阶段目标

在 Phase 1 已完成的老人端需求提交闭环上，继续增强“能说明白、后台能看懂、社区能处理”的能力。

Phase 2 不重做首页和提交页风格，继续沿用 Phase 1 的 D 首页和 A 表单视觉。

## 必须保持

- 首页仍为“安靖社区 · 服务台”。
- 提交页仍为标准表单布局。
- 不恢复复杂二级分类。
- 不做在线支付。
- 不做真实派单承诺。

## Phase 2 功能范围

### 1. 语音增强

- 语音识别失败时显示明确提示。
- 保留手动输入兜底。
- 在工单中标记“有语音说明”。
- 后续可模拟保存原始语音留言。

### 2. 图片增强

- 上传图片后显示缩略图或文件名。
- 工单详情显示图片数量。
- 后台可看到该工单是否有图片。

### 3. 后台简版工单看板

新增一个后台页面，给社区值班人员使用。

后台字段：

- 工单编号
- 老人姓名
- 事项类型
- 备注摘要
- 是否有语音
- 是否有图片
- 风险等级
- 推荐处理方
- 当前状态
- 创建时间

后台操作：

- 标记已电话联系
- 转社区认证服务者
- 转邻里互助
- 标记完成

### 4. 工单详情页

从“我的工单”或后台看板进入。

显示：

- 老人信息
- 事项类型
- 文字说明
- 语音识别文本
- 图片
- 紧急程度
- 电话联系要求
- 风险等级
- 推荐处理方
- 状态时间线

## 验收标准

- Phase 1 首页和提交页视觉不被破坏。
- 老人提交后能在后台看板看到工单。
- 后台可以修改工单状态。
- 我的工单页能看到状态变化。
- 图片上传后能看到预览或文件名。
- 语音失败时有文字输入兜底。
