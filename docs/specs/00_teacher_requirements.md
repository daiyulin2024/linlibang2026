# 00 老师要求整理

本项目按选项 B：公共服务创新路演准备。

## 评分重点

- 现场汇报：讲清“痛点-方案-价值”，问答时全组成员都熟悉项目。
- 展示海报：问题陈述、解决方案、技术流程、产品截图、3-5 个亮点、团队信息。
- 项目报告：目标用户、使用场景、问题定义、产品目标、方法描述、AI 使用过程、批判性反思。
- Python 源代码：本地可以运行，代码结构清楚，有 README。
- AI 协作日志：记录多轮对话、修改过程、采纳和拒绝理由，体现人主导 AI。

## 对本项目的转化

- 今日版必须有可运行网页 Demo。
- 代码必须体现 Python 使用，而不是只有设计图。
- 报告里要说明 Streamlit Demo 是课程展示版本，真实小程序和真实派单属于长期版。
- AI 协作痕迹要保留为项目资产。
